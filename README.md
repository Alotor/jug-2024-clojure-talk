# Java User Group (JUG) December 2024

From Java to Clojure (and not dying trying).

## Run main (Java version)

clojure -A:run

## Run main (Javascript version)

npx shadow-cljs watch app

Open the browser URL at http://localhost:8000

## Run tests

clojure -X:test

## Build

clojure -T:build ci
